Quais os comandos para configurar usuário e email no git?

git config --global user.name "nome sobrenome"

git config --global user.email "nome@sobrenome.com"

Qual a diferença entre git e github?

Git é um sistema de controle de versão.

GitHub cria ferramentas que usam Git.

O que faz o comando commit?

Adiciona mudanças, um save point do projeto.

O que faz o comando push?

Git push adiciona as suas modificações para o servidor, incluido-as no histórico do projeto.

O que faz o comando pull?

?Usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.
